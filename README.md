# hotelManagementSystem

## Project Description
A Java-based Hotel Management System developed as a university group project using JavaFX. The application provides a graphical user interface (GUI) that allows users to manage hotel bookings and related services efficiently. It displays reservation details and allows easy modification and management of bookings.

## Features
- **Reservation Services:**
  - Book a room, suite, or hall.
  - Order food from the hotel's restaurant.
  - Modify existing bookings (e.g., upgrade from room to suite).
- **Booking Details Display:**
  - Show reservation information including room number, number of nights, guest name, and booking cost.
- **Service Management:**  
  - Display additional hotel services available for each booking.
- User-friendly GUI built with JavaFX.
- Demonstrates teamwork and object-oriented programming concepts.
